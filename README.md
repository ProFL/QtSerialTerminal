# Simple Serial Terminal
A Simple Serial Terminal, originally built using Qt 5.7 meant to be simple, yet complete and multi platform compatible.
It was created to be used as a support tool in Microprocessors/Microcontrollers classes involving serial communication.

Currently the whole GUI is exclusively in Portuguese Brazilian, future localization to American English and Japanese is planned.
